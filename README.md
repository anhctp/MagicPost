# MagicPost

## Introduction
Delivery management system.
Features:

- Transaction points:
    + Direct transactions with customers.
    + Transactions with the nearest collection points.
- Gathering points:
    Transshipment of goods between transaction points and collection points.

## Flow

## Technology
- Front-end: NextJs
- Back-end: FastAPI
- Database: Mysql
